# domain

A Clojure library designed to create an HTTP REST API on top of a MongoDB data store.

## Usage

Include the library in your project. Instructions are [on Clojars](https://clojars.org/com.timezynk/domain).

## Development

To run the tests, do:

``` bash
lein test
```
